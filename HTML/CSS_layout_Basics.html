Getting Start with CSS Layout

layout တွေက css တွေမှာ tricky တွေအမျိုးမျိုးနဲ့အသုံးပြုနိုင်ပါတယ်။ browser တွေပေါ်မှာ layout တွေအမျိုးမျိုးနဲ့ကျွန်တော်တို့တွေတည်ဆောက်နိုင်ပါတယ်။ဒါပေမယ့်ကျွန်တော်တို့ websitesတွေ app တွေတည်ဆောက်တဲ့အခါမှာ layout တွေကအလွန်အရေးကြီးတဲ့အပိုင်းမှာပါဝင်တာပဲဖြစ်ပါတယ်။ကျွန်တော်တို့လေ့လာသွားရမှာကတော့ developer တွေ designer တွေအများဆုံးအသုံးပြုတဲ့ display mode , float တွေနေရာချထားဖို့ရန်အတွက်အသုံးပြုတဲ့ absolute , fix စတာတွေပဲဖြစ်ပါတယ်။ကျွန်တော်တို့အရင်သင်ခန်းစာတွေထဲမှာလည်း layout အကြောင်းအချို့ကိုလေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ Browser တွေမှာ default stylesheet တွေပါတယ်ဆိုတာကိုသိပြီးသားဖြစ်မှာပါ။ကျွန်တော်တို့ကအဲ့တာတွေကို user agent stylesheet လို့ခေါ်ပြီးတော့နောက်ထပ် stylesheet တွေထပ်ပြီးမသတ်မှတ်ခင်မှာအဲ့တာတွေကသတ်ရောက်နေမှာဖြစ်ပါတယ်။Browser က body ,heading, paragraph ,list စတဲ့ elements တွေကို margin, padding,linehight တွေ font-size value ဆိုတဲ့ default value တွေကိုသတ်မှတ်ပေးထားတာပဲဖြစ်ပါတယ်။popular web browser တွေဖြစ်တဲ့ chrome,firefox,safari, internet explorer စတဲ့ browser တွေမှာရှိတဲ့ default styling တွေကအနည်းငယ်ကွဲပြားမှုတွေရှိပြီးတော့ heading တွေ margin တွေစတာတွေမှာဖြစ်ပါတယ်။အဲ့တာတွေကိုရှင်းပြစ်ဖို့ရန်အတွက်အကောင်းဆုံးနည်းလမ်းကတော့ css reset ကိုအသုံးပြုချင်းပဲဖြစ်ပါတယ်။css reset ကအမျိုးမျိုးသော browser တွေမှာရှိတဲ့ margin,padding,line height,font-size စတဲ့ default styling တွေကိုဖယ်ရှားပေးနိုင်ပါတယ်။နောက်တစ်နည်းပြောရမယ်ဆိုရင်တော့ browser တွေအားလုံးကိုတူညိစွာအလုပ်လုပ်အောင်လုပ်ဆောင်ပေးတာပဲဖြစ်ပါတယ်။web designer တွေ web developer တွေကတော့မတူညီတဲ့ css reset နည်းလမ်းတွေကိုအသုံးပြုကြပါတယ်။ကျွန်တော်တို့လေ့လာသွားမှာကတော့ popular ဖြစ်တဲ့ method တစ်ခုဖြစ်တဲ့ normalize.css ပဲဖြစ်ပါတယ်။အဲ့တာကကျွန်တော်တို့ website ထဲမှာရှိတဲ့ default styling တွေကိုဖယ်ရှားပေးပါလိမ့်မယ်။ Web design တွေမှာ wrapper ကိုအများဆုံးအသုံးပြုကြပါတယ်။layout ရဲ့အလယ်မှာထားဖို့အတွက်နဲ့ device ရဲ့ viewport တွေပေါ်မှာမှီခိုပြီးတအားကျယ်သွားတာတွေ တအားကျဉ်းသွားတာတွေကိုထိန်းသိမ်းဖို့ရန်အတွက်အသုံးပြုနိုင်ပါတယ်။web designer တွေ web developer တွေကတော့ container တွေကိအသုံးပြုပြီးတော့ container တွေ wrapper တွေက layout တွေထဲမှာ element တွေကိုဖော်ပြတဲ့အခါမှာအသုံးပြုတဲ့ generics term တွေပဲဖြစ်ပါတယ်။

Sticky Footer

ဒီတစ်ခါမှာတော့ကျွန်တော်တို့ sticky footer တွေကိုဘယ်လိုတည်ဆောက်မယ်ဆိုတာကိုလေ့လာရမှာပဲဖြစ်ပါတယ်။web designs တွေမှာဆိုရင် footer တွေက page ရဲ့အောက်ခြေမှာ sticky အနေနဲ့တည်ရှိနေရာမှာဖြစ်ပါတယ်။ကျွန်တော်တို့ page မှာရှိတဲ့ content ကအောက်ကိုကျယ်နေတဲ့အချိန်ပဲဖြစ်ဖြစ် ကျဉ်းနေတဲ့အချိန်ပဲဖြစ်ဖြစ် footer ကတော့ page ရဲ့အောက်ခြေမှာရှိနေရမှာပဲဖြစ်ပါတယ်။footer တွေက page ရဲ့အောက်ခြေမှာမရှိပဲ အပေါ်ကိုတက်နေတာတွေကတော့ ကျွန်တော်တို့ layout problem တွေကြောင့်ပဲဖြစ်ပါလိမ့်မယ်။ကျွန်တော်တို့မှာ css တွေကိုအသုံး ပြုပြီး sticky footer တွေကိုတည်ဆောက်ဖို့အတွက်မတူညီတဲ့နည်းလမ်းတွေရှိပါတယ်။ကျွန်တော်တို့ပြောပြသွားမှာကတော့ popular ဖြစ်တဲ့နည်းလမ်းတစ်ခုဖြစ်ပါတယ်။အဲ့တာကတော့ css calculate function ကိုvh unit တွေနဲ့တွဲပြီးအသုံးပြုတဲ့နည်းလမ်းပဲဖြစ်ပါတယ်။အသုံးပြုပုံကတော့ ပထမကျွန်တော်တို့ရဲ့ html document ထဲမှာ footer ကို အခြားcontent တွေနဲ့ခွဲထုပ်ဖို့အတွက် အခြား content တွေကို div tag ထဲမှာထည့်ထားပေးရပါမယ်။အဲ့ဒီနောက်ကျွန်တော်တို့က CSS မှာအဲ့ဒီ div tagရဲ့ min-height ကို calc function နဲ့သတ်မှတ်ပေးထားရမှာပဲဖြစ်ပါတယ်။ဥပမာထဲမှာပိုပြီးလေ့လာနိုင်ပါတယ်။အဲ့ဒီထဲမှာပါတဲ့ vh ဆိုတာကတော့ view port ရဲ့ realtive unit ဖြစ်တဲ့ view port height ကိုဆိုလိုပါတယ်။1vh က 1% of viewport နဲ့တူညီပါတယ်။ calc function ဆိုတာကတော့ css ထဲမှာရှိတဲ့တန်ဖိုးတွေကိုသတ်မှတ်တဲ့အခါမှာသုံးနိုင်တဲ့ function ပဲဖြစ်ပါတယ်။နှုတ်ခြင်း ပေါင်းခြင်း မြှောက်ခြင်း စားခြင်းတွေပြုလုပ်နိုင်ပြီးတော့ result တွေကိုထုပ်ပြီးသတ်မှတ်ပေးတာဖြစ်ပါတယ်။ 


@media (min-width: 769px) {
.wrap {
    min-height: calc(100vh - 89px);
}
}

Controlling layout with CSS Display Modes

layout တွေကို CSS Display modes တွေနဲ့ဘယ်လိုထိန်းချုပ်ရမလဲဆိုတာတွေကိုကျွန်တော်တို့လေ့လာသွားရမှာဖြစ်ပါတယ်။အသုံးအများဆုံး CSS Display Modes တွေကတော့ block ,inline block ,inline စတာတွေပဲဖြစ်ပါတယ်။အဲ့တာတွေကြားထဲမှာရှိတဲ့ခြားနားချက်တွေကိုကျွန်တော်တို့ကနားလည်ထားရပါမယ်။ကျွန်တော်တို့ webpage ထဲမှာရှိတဲ့ content တွေ element တွေက rectangle box ထဲမှာတည်ရှိတာပဲဖြစ်ပါတယ်။ block level element တွေကတော့ content တွေကိုရိုးရှင်းစွာတစ်ခုချင်းစီ block ပြုလုပ်ပြီးဖော်ပြတာပဲဖြစ်ပါတယ်။အဲ့ဒီ content တွေကြားထဲမှာလည်း line break တွေပြုလုပ်ပေးပါတယ်။ inline element ကတော့ image တွေ link tag တွေ span tag စတာတွေကိုဖော်ပြတဲ့အခါမှာအသုံးပြုတာဖြစ်ပြီးတော့ layout ပေါ်မှာတော့ Line break တွေပြုလုပ်ပေးမှာမဟုတ်ပါဘူး။inline တွေကိုအသုံးပြုတဲ့အခါမှာ element တွေကတူညီတဲ့ line တစ်ခုထဲပေါ်မှာတည်ရှိမှာပဲဖြစ်ပါတယ်။inline အသုံးပြုထားတဲ့ element တွေကို width တွေ့height တွေ margin-top တွေ button တွေ သတ်မှတ်ပေးရင်လည်းအလုပ်လုပ်မှာမဟုတ်ပါဘူး။ဒါပေမယ့် Inline element တွေကတစ်ခုနဲ့တစ်ခု float ပြုလုပ်ပေးနိုင်ပါတယ်။ 



.name,
.main-nav,
.main-nav li {
display: inline-block;
}

Layout with Float Property

float တွေဟာ css layout တွေမှာအသုံးပြုမှုအများဆုံး tools တွေပဲဖြစ်ပါတယ်။layout တွေမှာဆိုရင် Popular အဖြစ်ဆုံး Method တစ်ခုလည်းဖြစ်ပါတယ်။အဲ့ဒီအတွက်ကြောင့်ကျွန်တော်တို့က float ရဲ့မူလ concept တွေကိုနားလည်ထားရပါမယ်။websites တွေအများစုရဲ့ layout တွေထဲမှာ image တွေ content တွေရဲ့ပတ်လည်မှာ float ဖြစ်နေတဲ့ text paragraph ကိုကျွန်တော်တို့မကြာခနမြင်ဘူးပါလိမ့်မယ်။float တွေဟာ inline-block method နဲ့ပုံစံတူပြီးတော့ကျွန်တော်တို့တွေ အလျှားလိုက် navigation တွေ menu တွေ column တွေကိုတည်ဆောက်နိုင်တာပဲဖြစ်ပါတယ်။ အဲ့ဒီနှစ်ခုရဲ့အကြီးမားဆုံးခြားနားချက်ကတော့ float element တွေဟာပုံမှန် document တွေရဲ့အပြင်ဘက်ကိုပါယူပြီးfloat လုပ်နိုင်ပါတယ်။ပုံမှန်အားဖြင့် HTML မှာ element တွေကိုအစဉ်လိုက်စီပြီးတော့တည်ဆောက်ပေးပါတယ်အဲ့တာကို normal document flow လို့ခေါ်ပါတယ်။အဲ့ဒီ document flow တွေကိုကျွန်တော်တို့က float တွေသုံးပြီးပြောင်းလဲနိုင်ပါတယ်။ float တွေကိုအသုံးပြုတဲ့အခါမှာ element တွေကိုကျွန်တော်တို့ရွှေ့ထားချင်တဲ့နေရာကိုရွှေ့လို့ရပါတယ်။ဥပမာကျွန်တော်တို့ ပုံကိုရွှေ့ပြီးထားမယ်ဆိုရင်တော့စာသားတွေကပုံရဲ့ဘေးပတ်လည်မှာပေါ်နေမှာပဲဖြစ်ပါတယ်။အဲ့တာတွေကိုတော့ကျွန်တော်တို့က padding တွေ margin တွေသတ်မှတ်ပြီးခြားနိုင်ပါတယ်။ ကျွန်တောတို့တွေက inline-block method တွေကို layout တွေရဲ့ navigation items တွေကို side by side ထားဖို့အတွက်နဲ့ရိုးရှင်းတဲ့ two-column layout တွေကိုတည်ဆောက်ဖို့အတွက်အသုံးပြုနိုင်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့ center နဲ့ vertical alignment တွေကိုထိန်းညှိဖို့လိုအပ်တဲ့အခါတွေမှာလည်းအသုံးပြုနိုင်ပါတယ်။content တွေကို element ရဲ့ဘယ်ဘက်သို့မဟုတ်ညာဘက်ကို flow ပြုလုပ်ချင်တဲ့အခါတွေမှာလည်း float ကိုအသုံးပြုပါတယ်။float တွေက elements တွေကြားထဲမှာ default horizontal white space တွေကိုမတည်ဆောက်နိုင်ပါဘူး။ကျွန်တော်တို့က container တွေရဲ့အလုပ်လုပ်ပုံတွေကိုလည်းသတိပြုမိဖို့လိုအပ်ပါတယ်။

နောက်ထက်ရိုးရှင်းပြီး flexable ဖြစ်တဲ့နည်းလမ်းတွေကတော့element တွေကို overflow ပြုလုပ်ပြီး hidden မှာauto ပြုလုပ်ပေးထားမှာဖြစ်ပါတယ်။အဲ့တာကကျွန်တော်တို့ layout ထဲမှာ element တွေကို fixed ဖြစ်အောင်ပြုလုပ်ပေးနိုင်ပါတယ်။ နောက်ထပ်နည်းလမ်းကတော့ float ကို css pseudo element တွေနဲ့ clear ပြုလုပ်ပေးတာပဲဖြစ်ပါတယ်။အဲ့တာအတွက်ကျွန်တော်တို့က clearfix ဆိုတဲ့ method ကိုအသုံးပြုပြီးတော့အဲ့ဒီထဲမှာပါတဲ့ content ဆိုတာကတော့CSS တွေကြားထဲမှာရှိတဲ့ blank pseudo element တွေကို generate လုပ်ပေးဖို့အတွက်ပဲဖြစ်ပါတယ်။ display မှာ table value ကိုအသုံးပြုတာကတော့ pseudo element ကို block level table အဖြစ်ဖော်ပြဖို့ရန်အတွက်ဖြစ်ပါတယ်။နောက်ထပ် clear ဆိုတဲ့ decoration မှာတော့ clear property က float နဲ့တိုက်ရိုက်ဆက်နွယ်နေပြီးတော့အရေးကြီးတာတွေကိုထိန်းချုပ်နိုင်တာပဲဖြစ်ပါတယ်။
